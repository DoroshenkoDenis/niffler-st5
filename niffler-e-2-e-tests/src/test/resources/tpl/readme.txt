Шаблон FreeMarker, используемым для отображения информации о SQL-запросах в отчетах Allure.

Основные элементы этого шаблона:

1) Импорт переменной data:
<#-- @ftlvariable name="data" type="guru.qa.niffler.data.repository.logging.SqlRequestAttachment" -->
Это комментарий FreeMarker, который указывает, что в шаблоне будет использоваться переменная data типа SqlRequestAttachment.

2) Подключение библиотек:
Шаблон подключает jQuery, Bootstrap и библиотеку подсветки синтаксиса Highlight.js.
Эти библиотеки используются для улучшения внешнего вида и форматирования отображаемой информации.

3) Отображение SQL-запроса:
<pre><code>${data.sql}</code></pre>
Этот фрагмент отображает текст SQL-запроса, взятый из поля sql объекта SqlRequestAttachment.
Использование тегов <pre> и <code> обеспечивает предварительное форматирование текста и подсветку синтаксиса SQL.
Таким образом, этот шаблон FreeMarker определяет, как будет отображаться информация о SQL-запросах в отчетах Allure. Он использует внешние библиотеки для улучшения визуального представления данных и обеспечивает читабельность и подсветку SQL-кода.
Этот шаблон используется классом AllureAppender, который прикрепляет объекты SqlRequestAttachment к отчетам Allure. Таким образом, при генерации отчетов Allure, информация о SQL-запросах будет отображаться в соответствии с этим шаблоном.
